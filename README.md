# *DevDrop - AI-Powered Recruitment Platform*

## *📌 Purpose of the Application*
DevDrop is a modern, AI-enhanced recruitment platform designed to bridge the gap between talented candidates and innovative companies. It streamlines the hiring process by integrating video introductions and intelligent automated analysis.

### *How It Works?*
- *Candidates* record 30-second video pitches and apply to jobs with a single click.
- *Recruiters* post jobs and receive an automatically ranked list of applicants.
- *AI Engine* analyzes candidate submissions (text + skills) to provide instant "Smart Scores" and summaries.

### *How It Helps the Industry?*
- *Eliminates Resume Fatigue*: Recruiters watch engaging video intros instead of reading endless text.
- *Unbiased Automatic Ranking*: Candidates are ranked by skill match and potential, not just keywords.
- *Instant Feedback Loop*: Real-time tracking keeps everyone informed.

---

## *🌟 Features*

✔ *Video Self-Introductions* - Candidates can record and upload video pitches directly in the browser.  
✔ *Smart Applicant Ranking* - Applicants are automatically sorted by relevance using AI.  
✔ *AI Insights Engine* - Google Gemini analyzes profiles to generate "Match Scores", summaries, and key strengths.  
✔ *Instant Video Playback* - Zero-latency video streaming for recruiters (Simulated CDN).  
✔ *Job Workflow Management* - Drag-and-drop style status updates (Shortlisted, Interview, Hired).  
✔ *Glassmorphic UI* - A premium, dark-mode enabled interface built with React & Tailwind.  
✔ *Secure Authentication* - Role-based access control via Clerk.

## *🛠️ Tech Stack & APIs Used*

- *Frontend:* React.js (Vite), Tailwind CSS, ShadCN UI  
- *Backend:* Node.js, Express.js  
- *Database:* Firebase Firestore  
- *AI Engine:* Google Gemini (Generative AI)  
- *Storage:* Local Storage (Demo) / Firebase Storage (Prod)  
- *Authentication:* Clerk

## *📖 How to Fork/Clone the Project?*

1. *Fork the Repository* - Click the "Fork" button on GitHub.
2. *Clone the Repository*
   bash
   git clone https://github.com/Harshal7890/DevDrop-Commited-GFG-PCCOE.git
   cd devdrop
   
3. *Install Dependencies*
   bash
   ### Client
   cd client && npm install
   ### Server
   cd ../server && npm install
   
4. *Run Locally*
   bash
   ### Start Client
   npm run dev
   ### Start Server
   npm start
